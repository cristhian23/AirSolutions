# AirSolutions

Proyecto ASP.NET Core con Bootstrap (sin Razor). Sirve archivos estáticos desde `wwwroot`.

## Ejecutar

```bash
cd AirSolutions
dotnet run
```

Abrir http://localhost:5158

## Subir a GitHub

1. En Cursor: abre el panel **Source Control** (Ctrl+Shift+G).
2. Haz clic en **"Publish to GitHub"** (o "Publicar en GitHub").
3. Elige **crear un repositorio nuevo**, nombre: `AirSolutions`, y tu cuenta.
4. Después del primer push, los cambios se suben desde el mismo panel con **Sync** o **Push**.

Desde terminal (si ya creaste el repo en github.com):

```bash
git remote add origin https://github.com/TU_USUARIO/AirSolutions.git
git branch -M main
git push -u origin main
```
